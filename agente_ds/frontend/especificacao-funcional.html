2. Agente 1 — Qualificação de Ofertas

Análise inteligente de RFPs para scoring e recomendação Go/No-Go

▼
01

Qualificação de Ofertas

Avalia oportunidades comerciais, produz scoring multicritério e recomendação Go/No-Go

📥 Inputs

  • Documento da Oportunidade — RFP, RFI ou Edital (PDF, DOCX, XLSX — máx. 50MB)
  • Nome da Oportunidade — Identificação textual do deal
  • Cliente — Empresa demandante
  • Segmento / Vertical — Energia, Financeiro, Governo, Telecom, Indústria, Saúde, Transporte, Varejo
  • Tipo de Serviço — Desenvolvimento, Consultoria, Sustentação, Cloud, Data & IA, DS, Cyber
  • Observações — Contexto complementar

⚙️ Função

  • Extração de Requisitos — Parsing para identificar escopo, tecnologias, prazos
  • Scoring Multicritério — Portfólio 30%, Equipe 20%, Fit Comercial 25%, Risco 15%, Referências 10%
  • Análise de Aderência — Cruzamento com portfólio via RAG
  • Identificação de Riscos — Técnicos, comerciais e timeline
  • Recomendação Go/No-Go — Threshold configurável (padrão: 60)
  • Mapeamento de Competências — Skills requeridas vs disponíveis

📤 Outputs

  • Score de Qualificação — 0 a 100 com classificação visual
  • Recomendação Go/No-Go — Decisão com justificativa
  • Análise de Aderência — % requisitos cobertos
  • Riscos Identificados — Lista categorizada
  • Competências-Chave — Tags de skills
  • Recomendações Estratégicas — Abordagem e mitigação

🔧 Configuração Técnica

Modelo LLM:Claude 3.5 Sonnet
Temperatura:0.3 (precisa)
Max Tokens:4.096
KBs:RFPs, Portfólio, Cases
Chat:Sim — regenera outputs
Herança:→ Agente 2 e 3
📝

3. Agente 2 — Gerador de Template Kickoff (S2S)

Geração de material para reunião de kickoff Sell-to-Serve

▼
02

Gerador de Template — Kickoff Meeting (S2S)

Produz artefatos para reunião de kickoff: agenda, checklist e apresentação

📥 Inputs

  • Materiais de Referência — Proposta, SOW, Contrato (PDF, DOCX, PPTX)
  • Resultado Step 1 — Herança: score, riscos, competências
  • Data do Kickoff — Data planejada
  • Participantes Indra — Nome e papel
  • Objetivos Específicos — Pontos prioritários
  • Riscos / Atenção — Riscos identificados

⚙️ Função

  • Geração de Agenda — Blocos de tempo sugeridos (90 min)
  • Definição de Papéis — Mapeamento de participantes
  • Alinhamento de Escopo — Governança, timeline, milestones
  • Análise de Riscos S2S — Consolidação para execução
  • Geração de Checklist — Pré-requisitos do kickoff
  • Formatação Template — Padrão visual Indra (PPTX)

📤 Outputs

  • Agenda do Kickoff — Detalhada com responsáveis
  • Papéis Sugeridos — Composição do time
  • Checklist Pré-Kickoff — Status pré-requisitos
  • Artefato Gerado — Template_Kickoff_[Cliente].pptx
  • Pontos de Alinhamento — Escopo e governança
  • Template de Ata — Estrutura pré-formatada

🔧 Configuração Técnica

Modelo LLM:GPT-4o (OpenAI)
Temperatura:0.5 (criativa)
Max Tokens:8.192
KBs:Portfólio, Templates, Cases
Formato:PPTX (padrão Indra)
Duração:90 minutos
Herança:← Ag.1 → Ag.3
💰

4. Agente 3 — Gerador de Ofertas Ballpark

Dimensionamento de time, custos e precificação preliminar

▼
03

Gerador de Ofertas Ballpark

Estimativa de custos, dimensionamento e precificação com 3 cenários

📥 Inputs

  • Tabela de Rates — Preços por perfil (XLSX, CSV)
  • Dados herdados (Steps 1+2) — Qualificação + Kickoff
  • Duração Estimada — Meses do projeto
  • Modelo de Contratação — T&M, Preço Fixo, Outcome, Híbrido, Body Shop
  • Localidade — Remoto, Híbrido, Presencial, Near/Offshore
  • Tecnologias / Stack — Lista requerida
  • Margem Alvo (%) — Lucro desejado
  • Restrições Comerciais — Limitações orçamentárias

⚙️ Função

  • Dimensionamento — Perfis, quantidades (30% Sr / 50% Pl / 20% Jr)
  • Cálculo de Custos — Pessoal + infra + ferramentas + contingência (10%)
  • Overhead Gerencial — Fator configurável (12%)
  • Precificação 3 cenários — Margens distintas
  • Calibração Histórica — RAG com projetos similares
  • Projeção de Impacto — Vertical/horizontal/consolidado
  • Roles Gerenciais — SM ≥8 FTEs, TL ≥4 FTEs

📤 Outputs

  • Dimensionamento — Tabela de perfis e FTEs
  • Estimativa de Custos — Breakdown completo
  • 3 Cenários — Conservador, Recomendado, Agressivo
  • Indicadores-Chave — Preço (R$), Margem, FTEs, Duração
  • Impacto Operações — Projeção de Venda, Contratação, Margem
  • Insight Estratégico — Posicionamento da oferta

🔧 Configuração Técnica

Modelo LLM:Claude 3.5 Sonnet
Temperatura:0.2 (alta precisão)
Max Tokens:8.192
KBs:Dimensionamentos, Rates
Margem Min:15%
Margem Padrão:25%
Margem Max:35%
Contingência:10%
Overhead:12%
Horas/Mês:168h
Produtividade:85%
Moeda:BRL (R$)
